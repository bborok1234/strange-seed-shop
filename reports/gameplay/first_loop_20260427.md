# 첫 플레이 루프 검증 보고서: 2026-04-27

## 범위

Milestone 3의 최소 루프를 구현했다.

포함:

- 스타터 씨앗 선택
- 첫 밭 자동 심기
- 탭 성장 가속
- 성장 완료 후 수확
- 첫 생명체 도감 등록
- 첫 도감 보상
- 두 번째 밭 업그레이드
- 첫 원정 시작/완료/보상
- 15분 이상 이탈 후 오프라인 보상 계산

## 검증

### 콘텐츠 참조 검증

명령:

```bash
npm run check:content
```

결과:

- JSON 파싱 성공
- manifest asset 수: 20
- 누락된 asset 참조: 0
- 누락된 manifest 파일: 0

### 첫 루프 데이터 검증

명령:

```bash
npm run check:loop
```

검증 기준:

- 스타터 씨앗 3개 존재
- 첫 씨앗 성장 시간이 90초 이하
- 첫 씨앗이 생명체 pool을 가짐
- 첫 수확 + 첫 도감 보상으로 첫 업그레이드 비용 충족
- 첫 원정이 10분 이하

### 빌드 검증

명령:

```bash
npm run build
```

결과:

- TypeScript 통과
- Vite production build 성공

### 로컬 서버

명령:

```bash
curl -I -s http://127.0.0.1:3000
```

결과:

- HTTP 200 OK

## 남은 리스크

- 실제 브라우저 클릭 플로우 시각 QA는 아직 별도 자동화하지 않았다.
- 오프라인 보상은 localStorage 기반이며 서버 시간 검증은 Phase 1 범위다.
- 첫 루프는 기능 검증용 최소 구현이며 밸런스/튜토리얼 카피는 추가 조정이 필요하다.
